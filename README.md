# Terminal chat

**System requirements**
* nodejs >= 12
* npm >= 6.12
* openssl >= 1.1.1

## Install
Need run commands in the terminal for Linux
```bash
wget https://github.com/reptily/openchat/archive/master.zip
unzip master.zip
cd openchat-master
npm install
npm run certification
```

### Start Client
```bash
npm run client
```

### Start Server
```bash
npm run server
```

### Config

All necessary configurations are in th file *config.js*

### Generation certification
```bash
npm run certification
```
